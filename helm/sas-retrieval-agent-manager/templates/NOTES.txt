Thank you for installing the SAS Retrieval Agent Manager Helm chart!

NOTE: This chart includes post-install hooks that may take several minutes to complete.
      When using --wait, ensure you set an adequate timeout (e.g., --timeout 10m).

You can now access your environment at:

  {{- $domain := .Values.ingress.domain -}}
  {{- $ui_ingress := (index .Values "ingress" "ui") -}}

  {{- $globalPath := index $ui_ingress.paths 0 }}

  {{- $localHost := "" }}
  {{- $localExtractedPath := "" }}

  {{- if and $domain $ui_ingress }}
    {{- $localHost := $domain }}
    {{- $localHostPath := index ($ui_ingress).paths 0 }}
    {{- $localExtractedPath := $localHostPath }}
  {{- end }}

  https://{{ if .Values.ingress.enabled | default false }}{{ .Values.ingress.domain }}{{ $globalPath }}{{ else }}{{ $localHost }}{{ $localExtractedPath }}{{ end }}

Login with:

  Username: {{ .Values.users.application.admin.username }}

  To get the app password, run the following command:
  - Powershell: kubectl get secret -n retagentmgr {{ printf "%s-appadmin-secret" (include "keycloak.fullname" .) }} -o jsonpath="{.data.password}" | ForEach-Object { [System.Text.Encoding]::UTF8.GetString([System.Convert]::FromBase64String($_)) }
  - Bash: kubectl get secret -n retagentmgr {{ printf "%s-appadmin-secret" (include "keycloak.fullname" .) }} -o jsonpath="{.data.password}" | base64 --decode && echo
