Thank you for installing the SAS Retrieval Agent Manager Helm chart!

You can now access your environment at:

  {{- $app := index .Values "sas-retrieval-agent-manager-app" -}}
  {{- $ingress := (index $app "ingress") -}}

  {{- $globalPath := (index $ingress.paths 0).path }}
  {{- $globalExtractedPath := regexFind "^/[^(/]+" $globalPath }}

  {{- $localHost := "" }}
  {{- $localExtractedPath := "" }}

  {{- if and $ingress.hosts (gt (len $ingress.hosts) 0) }}
    {{- $localHost := (index $ingress.hosts 0).host }}
    {{- $localHostPath := (index (index $ingress.hosts 0).paths 0).path }}
    {{- $localExtractedPath := regexFind "^/[^(/]+" $localHostPath }}
  {{- end }}

  https://{{ if .Values.global.ingress.enabled | default false }}{{ .Values.global.domain }}{{ $globalExtractedPath }}{{ else }}{{ $localHost }}{{ $localExtractedPath }}{{ end }}

Login with:

  Username: {{ .Values.global.configuration.keycloak.appAdmin }}

  To get the app password, run the following command:
  - Powershell: kubectl get secret -n retagentmgr keycloak-appadmin-secret -o jsonpath="{.data.password}" | ForEach-Object { [System.Text.Encoding]::UTF8.GetString([System.Convert]::FromBase64String($_)) }
  - Bash: kubectl get secret -n retagentmgr keycloak-appadmin-secret -o jsonpath="{.data.password}" | base64 --decode && echo

For user management, access Keycloak at:

  {{- $kc := index .Values "keycloak" -}}
  {{- $ingress := (index $kc "ingress") -}}

  {{- $adminHost := $ingress.admin.host }}
  {{- $adminPath := $ingress.admin.path }}

  https://{{ if .Values.global.ingress.enabled | default false }}{{ .Values.global.domain }}{{ $adminPath }}{{ else }}{{ $adminHost }}{{ $adminPath }}{{ end }}

Login with:

  Username: {{ .Values.global.configuration.keycloak.keycloakAdmin }}

  To get the keycloak password, run the following command:
  - Powershell: kubectl get secret -n retagentmgr keycloak-admin-secret -o jsonpath="{.data.password}" | ForEach-Object { [System.Text.Encoding]::UTF8.GetString([System.Convert]::FromBase64String($_)) }
  - Bash: kubectl get secret -n retagentmgr keycloak-admin-secret -o jsonpath="{.data.password}" | base64 --decode && echo

For further help, check your documentation or contact your SAS Administrator.
